# 贡献指南

感谢你对 SZYX AI 项目的关注！我们欢迎任何形式的贡献。

## 如何贡献

### 报告 Bug

1. 使用 [GitHub Issues](https://github.com/2919671430/szyx/issues) 提交 Bug
2. 请使用 Bug 报告模板
3. 包含复现步骤、预期行为和实际行为
4. 如果可能，附上日志截图

### 提交功能建议

1. 在 Issues 中使用 "功能建议" 标签
2. 详细描述你希望的功能
3. 说明使用场景

### 提交代码

1. Fork 本项目
2. 创建你的功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'feat: add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

## 开发规范

### 代码风格

- 遵循 Google Java Style Guide
- 使用 4 空格缩进
- 类名使用 PascalCase
- 方法名使用 camelCase
- 常量使用 UPPER_SNAKE_CASE

### Commit 消息格式

```
<type>(<scope>): <subject>

<body>

<footer>
```

类型（type）：
- `feat`: 新功能
- `fix`: 修复 Bug
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 重构
- `test`: 测试相关
- `chore`: 构建/工具相关

### 分支命名

- `feature/xxx`: 新功能
- `fix/xxx`: Bug 修复
- `docs/xxx`: 文档更新

## 开发环境

1. Android Studio Hedgehog 或更高版本
2. JDK 11
3. Android SDK 34

## 问题交流

如有任何问题，欢迎通过 Issues 交流。
