# SZYX AI 使用手册

---

## 一、应用简介

SZYX AI 是一款支持**多种云端 AI 模型**的角色扮演聊天应用。你可以创建自定义角色，与 AI 进行沉浸式对话，支持长期记忆、世界书知识库、养成数值系统等高级功能。

### 核心特性

- **多模型切换**：支持小米 mimo、DeepSeek、自定义 API 等多种云端模型，随时切换不丢失对话
- **角色卡系统**：自由创建角色人设、世界观、输出风格
- **双记忆体系**：短期记忆（最近 N 轮）+ 长期记忆（自动提取关键剧情）
- **世界书知识库**：关键词触发的世界观设定注入
- **养成数值系统**：AI 自动解析数值变化，独立存储
- **模板一键套用**：5 套预设模板，新手秒出作品

## 二、首次使用

### 2.1 启动应用

首次打开应用会进入**模型配置页面**：

- 直接点击「跳过」进入主界面，后续在设置中配置 API Key 即可

### 2.2 配置 API Key

1. 进入主界面后，点击右上角齿轮图标进入**设置**
2. 找到「API 设置」区域
3. 选择你想使用的 AI 服务（小米 mimo、DeepSeek、自定义 API）
4. 在「API Key」输入框中填入你的 Key
5. 点击「保存设置」

> 配置完成后，所有新对话默认使用你选择的 AI 模型。

---

## 三、角色创建

### 3.1 快速创建（模板方式）

1. 在主界面点击右下角 **+** 按钮
2. 进入角色编辑页面后，点击顶部「**一键套用模板**」
3. 从 5 套预设模板中选择一个：
   - **温柔女友**：温柔体贴的女友角色
   - **古风仙侠**：仙侠世界观角色扮演
   - **赛博朋克**：未来科幻 AI 助手
   - **治愈小动物**：可爱宠物互动
   - **悬疑侦探**：推理悬疑角色
4. 套用后可自由修改任何字段
5. 点击「保存」

### 3.2 自由创建

所有字段均为**可选**（除角色名称外），你可以只填写想写的部分：

| 字段 | 说明 | 示例 |
|------|------|------|
| 角色名称 | 必填 | 小月 |
| 性格特征 | 逗号分隔 | 温柔、聪慧、偶尔傲娇 |
| 系统提示词 | 角色的核心人设描述 | 你是一个来自古代的仙女... |
| 世界观设定 | 自由文本，描述故事背景 | 这是一个修仙世界... |
| 角色简介 | 角色的背景介绍 | 小月是青云宗的弟子... |
| 开场白 | 角色的第一句话 | 你好呀，你是新来的弟子吗？ |
| 输出风格 | AI 回复的风格要求 | 用古风文雅的语言... |
| 标签 | 逗号分隔，用于分类 | 仙侠,女性,古风 |

### 3.3 高级设置

#### 约束强度

控制 AI 遵守设定的严格程度：

- **宽松**：AI 有较大自由发挥空间，适合轻松聊天
- **标准（默认）**：保持人设一致性，不擅自改变剧情
- **严格**：严格按照设定行动，不添加未定义内容

#### 功能开关

| 开关 | 默认 | 说明 |
|------|------|------|
| 长期记忆 (LTM) | 开启 | 自动提取关键剧情、关系变化存入长期记忆 |
| 世界书知识库 | 开启 | 关键词触发世界观设定注入 |
| 养成数值系统 | 关闭 | 开启后 AI 会输出数值变化（如 [好感+5]） |

#### 短期记忆轮数

滑块调节，范围 5-100 轮，默认 20 轮。数值越大，AI 能参考的近期对话越多，但会占用更多 Token。

### 3.4 导入/导出

- **导入 JSON**：从剪贴板导入角色卡（兼容 SillyTavern 格式）
- **导出 JSON**：将当前角色导出为 JSON 到剪贴板

## 四、对话聊天

### 4.1 创建会话

1. 点击角色卡片进入会话列表
2. 点击右下角 **+** 创建新会话
3. 输入会话名称（如「第一章」「日常聊天」）

### 4.2 发送消息

在底部输入框输入文字，点击发送按钮即可。AI 会**流式输出**回复内容，你可以实时看到生成过程。

### 4.3 切换模型

在聊天界面顶部，角色名右侧有一个**模型标识**（显示「小米」或「本地」）：

1. 点击该标识
2. 在弹出的选择框中选择目标模型
3. 切换后当前会话后续消息使用新模型，**对话历史完整保留**

> 切换模型不会丢失任何对话内容、角色设定或数值进度。

### 4.4 查看数值状态

如果角色开启了「养成数值系统」，聊天界面顶部会显示当前数值条：

```
好感: 75/100 | 修为: 30/100 | 金币: 500
```

数值会随 AI 回复自动更新（AI 输出 `[好感+5]` 这样的标记时自动解析）。

### 4.5 停止生成

生成过程中可以随时点击发送按钮（此时变为停止状态）中断当前生成。

## 五、记忆系统

### 5.1 短期记忆 (STM)

- 默认保留最近 **20 轮**对话
- 可在角色编辑页面通过滑块调整（5-100 轮）
- 超出轮数的旧消息不会丢失，只是不发送给 AI

### 5.2 长期记忆 (LTM)

- **自动提取**：每积累约 30 条消息，系统自动调用 AI 提取关键信息
- **分类存储**：提取的记忆按标签分类：
  - `[剧情]`：关键剧情发展
  - `[好感]`：角色关系变化
  - `[抉择]`：用户做出的重要选择
  - `[事件]`：重要事件记录
- **永久挂载**：长期记忆在每次对话中都会注入到 Prompt 头部，不会被截断
- **可关闭**：在角色编辑页面关闭「长期记忆」开关

> 长期记忆解决了「AI 聊久了忘记之前发生了什么」的问题。

### 5.3 对话压缩

当 Token 使用超过上下文窗口的 80% 时，系统会自动将旧消息压缩为摘要，腾出空间继续对话。

## 六、世界书知识库

### 6.1 什么是世界书

世界书是一个**按需触发**的知识库。你为角色设定大量世界观信息，但只有当用户消息中出现相关关键词时，对应内容才会注入给 AI。

这样做的好处：
- 不会一次性占用大量 Token
- 宏大世界观也能高效使用
- AI 只在需要时获取相关背景

### 6.2 使用方式

1. 编辑角色时，确保「世界书知识库」开关已开启
2. 在角色的「世界观设定」字段中自由撰写世界观背景
3. 世界书会根据用户消息中的关键词自动匹配并注入

> 当前版本的世界书基于角色的世界观设定字段自动工作。后续版本将支持独立的世界书条目管理界面。

### 6.3 触发规则

- 用户消息中包含关键词时触发
- 多个匹配按优先级排序
- 触发的内容注入到 Prompt 的「世界设定」层

## 七、养成数值系统

### 7.1 开启方式

编辑角色 → 开启「养成数值系统」开关 → 保存

### 7.2 工作原理

1. 系统会在 Prompt 中添加数值变化指令
2. AI 在回复中标记数值变化，例如：
   - `你送给她一朵花，她开心地笑了 [好感+5]`
   - `你消耗了10点灵力释放法术 [灵力-10]`
   - `你获得了100枚金币 [金币+100]`
3. 系统自动解析这些标记，更新数据库
4. 聊天界面顶部实时显示当前数值

### 7.3 数值规则

- **自动创建**：首次出现的数值名自动创建，默认值 50，上限 100
- **支持增减**：`[好感+5]` 增加 5，`[修为-3]` 减少 3
- **自动封顶**：数值不会超过最大值或低于最小值（默认 0-100）
- **显示格式**：`数值名: 当前值/最大值`

### 7.4 适用场景

- 好感度养成
- RPG 属性系统（修为、体力、魔力等）
- 经济系统（金币、资源等）
- 任何需要数值追踪的场景


- 如果会话明确指定了模型（通过聊天界面切换），使用该模型
- 否则使用设置中配置的默认 AI 服务

## 九、Prompt 分层架构（技术说明）

每次对话的 Prompt 按以下顺序组装，**优先级从高到低**：

```
1. 系统骨架（内置规则，不可修改，永不截断）
2. 用户设定（角色人设 + 世界观，永不截断）
3. 长期记忆 LTM（关键剧情/抉择/关系，永不截断，上限 1500 Token）
4. 世界书（关键词触发的设定，按优先级截断，上限 2000 Token）
5. 养成数值（当前数值状态）
6. 短期记忆 STM（最近 N 轮对话，从最旧的开始截断）
7. 用户最新消息（永不截断）
```

**截断铁则**：只裁剪老旧历史对话，核心设定和记忆永远保留。

## 十、常见问题

### Q: API 调用失败怎么办？

1. 检查 API Key 是否正确
2. 检查网络连接
3. 确认 API 服务是否正常运行

### Q: 切换模型后对话会丢失吗？

不会。所有对话历史、角色设定、数值进度都与模型无关，切换模型只是换了推理引擎。

### Q: 长期记忆什么时候提取？

每积累约 30 条新消息自动提取一次。提取过程在后台进行，不影响当前对话。

### Q: 如何关闭所有高级功能？

编辑角色 → 关闭所有开关（长期记忆、世界书、养成数值）→ 约束强度设为「宽松」。这样就是一个纯聊天模式。

### Q: 支持哪些 AI 服务？

目前支持：
- 小米 mimo-v2.5
- DeepSeek
- 任何兼容 OpenAI 格式的自定义 API

---

### Q: 如何创建 RPG 养成角色？

1. 创建角色，填写世界观和人设
2. 开启「养成数值系统」
3. 在系统提示词中定义初始数值：`你有以下属性：修为=10，金币=100，体力=80`
4. 约束强度设为「标准」或「严格」
5. 开始对话，AI 会自动追踪数值变化
